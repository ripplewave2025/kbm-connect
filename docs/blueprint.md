# **App Name**: KBM Connect

## Core Features:

- Member Login: Secure member login with role-based access control.
- Mission Overview: Display a concise summary of KBM's mission and history.
- Video Gallery: Placeholder section for embedded videos showcasing KBM activities and success stories.
- Latest News: Display latest news about KBM and DCT partnership
- AI-Powered Summaries: Generate concise summaries of news articles, donation impact reports, or project updates, providing users with quick insights, using an AI summarization tool.
- Get Involved: Contact information and donation details to support KBM's mission.

## Style Guidelines:

- Primary color: Golden yellow (#FFC107) to represent warmth, enlightenment, and the Buddhist tradition.
- Background color: Very light desaturated yellow (#F8F9FA) to create a clean, calming backdrop.
- Accent color: Muted orange (#E0A200) to highlight important sections, call to action.
- Headline Font: 'Poppins', a geometric sans-serif for headlines. 
- Body Font: 'PT Sans', a humanist sans-serif for readability in longer text sections.
- Use clean, modern icons to represent different sections and actions.
- A clear and intuitive layout, optimized for easy navigation on both desktop and mobile devices.
